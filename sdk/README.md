The official Rust SDK for Accudo.

## Usage

This SDK provides all the necessary components for building on top of the Accudo Blockchain. Some of the important modules are:

* `client` - Includes a [REST client](https://accudo.dev/nodes/accudo-api-spec#/) implementation
* `crypto` - Types used for signing and verifying
* `transaction_builder` - Includes helpers for constructing transactions
* `types` - Includes types for Accudo on-chain data structures

## Installing Rust SDK
Please refer to [Rust SDK Doc](https://accudo.dev/sdks/rust-sdk/) for details on how to install the Rust SDK.
